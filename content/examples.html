These examples help the developers understand how to use the framework. They also show the capabilities and potentials that can be employed in your projects. Moreover, the code is aimed to demonstrate and guide. Therefore, advanced simulation methods (e.g. Monte Carlo) are avoided in the examples. The source codes are found under the examples folder.
  • Simple Matrix demonstrates a set of operations with types and methods that helps getting started.

  • Dijkstra routing algorithm demonstrated wih an example.

  • QAM Modulation over a Gaussian noisy channel that is a classic digital communication scheme.

  • Convolutional Error Correcting Code for a Gaussian noisy channel to show how the encoder and BCJR decoder are used.

  • Root-Raised Cosine (RRC) Matched Filter for communication over a Gaussian noisy channel.

The test cases are also a learning material specially for those parts of Susa that have not been used in the examples.
